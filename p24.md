Generate a lightweight Employee Management system for admin users.

Requirements:

* create employee accounts
* assign employee role
* assign department
* assign shift start time
* list employees
* search/filter employees
* mark attendance entries as invalid
* maintain audit logs

Roles:

* Admin
* Employee

Admin capabilities:

* create employees
* manage employees
* view analytics
* configure office settings

Employee capabilities:

* check-in/check-out
* view own attendance

Use:

* PostgreSQL
* Express backend
* JWT auth
* React + TypeScript
* TailwindCSS

Keep the implementation:

* lightweight
* practical
* clean
* realistic

Do NOT implement overly complex enterprise permission systems.
